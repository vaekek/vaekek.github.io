Controlled Components

Ein Element wie ​<input>​ ​<textarea>​ ​<select>​ haltet sein eigenen State in plain HTML. Diese Elemente ändern den internen Wert, sobald jemand von aussen den Wert ändert. In React werden solche Elemente „uncontrolled component“ genannt, weil es sein eigenen State handlet. 
In React sollte man darauf achten, dass diese Elemente zu einem „controlled component“ werden.

Wie macht man das?
Man setzt den Wert des Elements. Der Wert wird meistens/immer bereits in einem state Property gespeichert. Man kann den Wert also von dort beziehen.

Beispiel:
class App extends Component{
...
render(){
const { searchTerm, list } = this.state;
return (
<div className="App">
<form>
<input
type="text"
value{searchTerm}
onChange={this.onSearchChange}
/>
</form>
...
</div>
);
}
}
