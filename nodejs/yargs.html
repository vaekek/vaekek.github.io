yargs

Installation
npm i yargs --save
yargs wird dafür verwendet Argumente über die cmd einfach zu interpretieren. Anstelle von schwierigem filtern, stellt yargs key:value Objekte zur Verfügung.
einfaches Beispiel:
$ ./plunder.js --ships=4 --distance=22
Plunder more riffiwobbles!
$ ./plunder.js --ships 12 --distance 98.7
Retreat from the xupptumblers!
const argv = require('yargs').argv
if (argv.ships > 3 && argv.distance < 53.5) {
console.log('Plunder more riffiwobbles!')
} else {
console.log('Retreat from the xupptumblers!')
}
Auch kann definiert werden, wieviele Argumente etc. mitgegeben werden müssen.
https://github.com/yargs/yargs/blob/HEAD/docs/examples.md
um zu definieren welche Argumente requiert sind kann man ein command chain einfügen:
const argv = yargs.commannd('add','Add a new note', {
title: {
describe: 'title of note',
//required or not
demand: true,
// shortcut
alias: 't'
},
body: {
describe: 'the body of the note',
demand: true,
alias: 'b'
}
}).help()
